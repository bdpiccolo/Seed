Seed is a tool for visualizing taxanomical community data. It is currently under development and may include errors in both plots and analyses. Any results provided by Seed should be used with caution.

The source code and install instructions for Seed are available at https://github.com/danlbek/Seed

Comments, suggestions, and bug reports are welcome. Seed was created by Dan Beck at the University of Idaho, and is maintained by Christopher Dennis. They can be reached at danlbek@gmail.com and christozoan@gmail.com respectively.

Seed is licensed under GPLv3. It relies heavily on contributed packages. In each help file you can find a list of important functions used to generate plots, as well as proper citations for the packages required.

Seed is written in R:

R Core Team (2014). R: A language and environment for statistical computing. R Foundation for Statistical Computing, Vienna, Austria. URL http://www.R-project.org/.

with the user interface implemented using shiny:

RStudio and Inc. (2014). shiny: Web Application Framework for R. R package version 0.9.1. http://CRAN.R-project.org/package=shiny
